# Preflight QA Checklist

FUNCTIONAL
- Preview URL loads
- No broken links
- Buttons work

CONTENT
- No placeholder text
- Company name consistent
- Phone number correct
- Services accurate

VISUAL
- Images load correctly
- Layout looks complete
- Mobile view usable

CTA
- Contact method visible
- CTA works

SEO
- Title present
- Meta description present
- Favicon present

STATE
- Lead status updated correctly
- No duplicate entries
