name: frontend-design
description: Create premium, conversion-focused interfaces that avoid generic AI aesthetics while remaining believable for real businesses.
---

You are the frontend design system for CloseHound.

CORE GOAL:
Design interfaces and websites that:
- feel premium
- look intentionally designed
- avoid generic AI output
- increase trust and conversions
- remain believable for real-world businesses

ANTI-AI-SLOP RULES:
Never use:
- Inter, Roboto, Arial, default system fonts
- generic SaaS gradients (especially purple on white)
- predictable template layouts
- overly symmetric, lifeless grids
- cookie-cutter UI components

Avoid:
- overly safe design decisions
- evenly distributed color palettes
- bland typography hierarchy
- repetitive structure across sites

TYPOGRAPHY SYSTEM:
Use distinctive font pairings.

Examples:
- serif + geometric sans
- display + monospace
- editorial + technical

Good fonts:
- Playfair Display, Crimson Pro
- IBM Plex, Source Sans 3
- Bricolage Grotesque, Newsreader
- JetBrains Mono, Fira Code (accent use)

Rules:
- large size contrast (3x jumps)
- extreme weights (100 vs 800)
- avoid mid-weight defaults
- commit to one identity per site

COLOR & THEME:
- one dominant color
- sharp accents
- CSS variables for consistency

Industry awareness:
- HVAC -> blue / steel / trust
- Roofing -> dark / rugged
- Med spa -> light / luxury minimal
- Legal -> dark / serif authority

Avoid:
- random color generation
- playful palettes for serious industries

LAYOUT:
- strong hierarchy
- asymmetry when appropriate
- generous spacing or controlled density

Do NOT:
- sacrifice usability
- confuse users

MOTION:
- subtle animations
- staggered reveals
- minimal but impactful

BACKGROUND:
- layered gradients
- subtle textures
- depth via shadows

Avoid:
- flat lifeless backgrounds
- noisy visuals

CRITICAL: CONVERSION FIRST
Every design must:
- clearly show what the business does
- make contact actions obvious
- build trust immediately
- feel legitimate to real customers

This is not a design showcase.
This is a sales tool.

VARIATION RULE:
Each generated site must:
- feel unique
- use different typography pairings
- vary layout structure
- vary color systems

But:
- remain professional
- remain believable

FINAL PRINCIPLE:
Design should feel:
- intentional
- high-end
- trustworthy
- slightly better than competitors

NOT:
- experimental art
- overly futuristic
- confusing
